# 🎓 Student Management System

## 👨‍💻 Author

**Jahid**,  **whatsapp: 8801309495010**





A simple **Student Management System** built with **Python Object-Oriented Programming (OOP)** concepts.

This project was created to practice core OOP principles such as **Abstraction, Inheritance, Method Overriding, Polymorphism, and Encapsulation**.

---

## 📌 Features

* Add new students
* Add new teachers
* Store marks for multiple subjects
* Calculate a student's average marks
* Display all students and teachers
* Menu-driven console application

---

## 🛠️ Technologies Used

* Python 3
* Object-Oriented Programming (OOP)

---

## 📂 Project Structure

```
Student Management System
│
├── Person (Abstract Class)
│
├── Student
│   ├── add_marks()
│   ├── calculate_average()
│   └── show_details()
│
└── Teacher
    └── show_details()
```
